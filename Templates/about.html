About The Project

Websale is a Django-based website selling platform concept.

This academic project demonstrates how Django can be used to build a service-based ecommerce website where customers browse website packages and contact the team for purchase or customization.

Project Vision

To create a beginner-friendly ecommerce website using Django and HTML templates for selling website development services.

Target Users

Small businesses, students, freelancers, and startups that need affordable websites.

Technology Stack

Django for backend logic, Django templates for frontend rendering, and custom CSS for styling.
